解决方案：
运行——>cmd
在控制台中运行以下命令设置java环境变量即可解决：
set java_home=C:Program FilesJavajdk1.8.0_05   (安装JDK的根目录)
set classpath=%JAVA_HOME%libtools.jar;%JAVA_HOME%libdt.jar;
set path=%JAVA_HOME%bin;
注：输入不区分大小写
